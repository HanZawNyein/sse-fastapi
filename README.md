## Realtime Log Streaming with FastAPI and Server-Sent Events

### Setup

#### Prerequisites

To run this program you will need python 3 (I used 3.8). Then run the following commands:

    - python3 -m venv venv
    - source venv/bin/activate
    - pip3 install -r requirements.txt

#### .ENV

    - LOGFILE=**.log

#### Running the code

    - python3 main.py
